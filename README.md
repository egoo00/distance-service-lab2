В. гр 334701 Вариант №1 Проверка работы программы проходит через URL ссылку: http://localhost:8080/distance?city1=<город1>&city2=<город2> Где <город1> и <город2> — названия городов, между которыми вы хотите рассчитать расстояние. Пример http://localhost:8080/distance?city1=Minsk&city2=Moscov
